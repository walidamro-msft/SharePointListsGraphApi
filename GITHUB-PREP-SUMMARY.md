## ?? Entra ID Configuration (From Screenshots)

Based on the provided screenshots, the following configuration is documented in `SETUP.md`:

### API Permissions
| Permission | Type | Admin Consent | Status |
|------------|------|---------------|--------|
| Sites.Read.All | Application | Yes | ? Granted |
| Sites.ReadWrite.All | Application | Yes | ? Granted |
| Sites.Selected | Application | Yes | ? Granted |
| User.Read | Delegated | No | ? Granted |

### Redirect URIs (Authentication)
1. `http://localhost:51189`
2. `https://login.microsoftonline.com/common/oauth2/nativeclient`
3. `msal[YOUR-CLIENT-ID]://auth`
4. `https://login.live.com/oauth20_desktop.srf`

### Client Secret
- Description: "Console App"
- Expiration: Configurable (e.g., 6-24 months)
- Secret ID: Generated by Azure
- Secret Value: Copy immediately after creation (masked in production)
